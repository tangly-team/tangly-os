## Purpose

The **tangly erp apps** bounded domain component provides regular business model abstractions for business applications written in Java.
The component is requires **Java 17 or higher**.

**Try it out**.

## Download and Documentation

The library can be included in Maven as

```xml

<dependency>
    <groupId>net.tangly.erp</groupId>
    <artifactId>apps</artifactId>
    <version>0.2.6</version>
</dependency>
```

The library can be included in Gradle as

```groovy
    implementation "net.tangly.erp:apps:0.2.6"
```

The documentation can be found under [Business documentation](https://tangly-team.bitbucket.io/docs/erp/agile/)

For any further question and discussion you can use the forum [tangly-OS-Components](https://groups.google.com/g/tangly-os-components)
