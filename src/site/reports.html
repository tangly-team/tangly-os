Introduction

This landing page provides links to various developer reports. Developers are encouraged to use the feature to improve the quality and legibility of their sourcecode.

The contents of the page is only available after a full build of the product using gradle. The command is gradle build test jacocoTestReport jacocoRootReport.

The repository is Bitbucket Repository.

The SonarQube Overview of last build is Quality Gate Last Build.

You shall run ./gradlew build jacocoTestReport jacocoRootReport -Pmode=int to generate all quality reports described below.

If you run gradle build only the coverage per module is generated.
